• HTC access
    • HTC can be accessed through `ssh htc.crc.pitt.edu`
    • Setting up ssh: https://www.tecmint.com/ssh-passwordless-login-using-ssh-keygen-in-5-easy-steps/
    • Setting up useful ssh config in `~/.ssh/config` (Note that keeping a connection alive will have the effect of not reloading the environment upon a new login)
      ```
      #keep connections alive
      ControlPersist 4h
      ControlMaster auto
      ControlPath /tmp/ssh_mux_%h_%p_%r


      ServerAliveInterval 60
      ServerAliveCountMax 100

      #set up an alias
      Host htc
      Hostname htc.csb.pitt.edu```

    • On the HTC side you can add useful navigation commands
      • Create director with your user name under `/zfs1/mchikina/`
      • Add the to your .bashrc for quick navigation to zfs and a nice prompt
        ```alias cdz='cd /zfs1/mchikina/<username>/'

        PS1='\[\033[01;38;5;198m\]\u@htc\[\033[01;37m\]:\w \$\[\033[00m\] '

        export PS1 ```

    • You can now put files in your zfs directory and organize it however you want
    • Ondemand HTC (which gives rstudio and jupyter interfaces) can be accessed here: https://crc.pitt.edu/ondemand
      • Read the various instruction as needed
      • Most useful libraries are already installed. If not you can install them.
      • Save your work as sessions expire!
